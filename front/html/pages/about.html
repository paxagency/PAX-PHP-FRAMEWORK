(URL Variable)

URL Variable Restrictions

This method creates requirements for variables within the URL after the route

$app->urlVars($urlCount, $strictMode);

$urlCount
Number of URL variables to allow
/one/two/three/etc...

$strictMode
default = All URL variables allowed
0 = Only allow route and number of URL variables
1 = Only allow route with URL variables
2 = Only allow route with all URL variables
